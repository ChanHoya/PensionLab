# Project Brief

## Vision
An AI-driven interactive pension planning responsive web application helper that aggregates and simulates all layers of pension in Korea.

## Goals
- Integrate National, Basic, Retirement, and Personal Pensions.
- Implement real-time simulations reflecting the 2026 pension reform.
- Provide expert content RAG advice.

## Non-Goals
- Real-time API integration with NPS in Phase 1 (MVP uses manual input).
